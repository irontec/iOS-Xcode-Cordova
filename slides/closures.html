Declaración de 'Closures' en Swift

Como variable:

var closureName: (parameterTypes) -> (returnType)

Como variable opcional:

var closureName: ((parameterTypes) -> (returnType))?

Como type alias:

typealias closureType = (parameterTypes) -> (returnType)

Como constante:

let closureName: closureType = { ... }

Como argumento de una llamada a una función:

func({(parameterTypes) -> (returnType) in statements})

Como parámetro de una función:

array.sort({ (item1: Int, item2: Int) -> Bool in return item1 < item2 })

Como parámetro de una función con tipados implícitos:

array.sort({ (item1, item2) -> Bool in return item1 < item2 })

Como parámetro de una función con tipado implícito en el retorno:

array.sort({ (item1, item2) in return item1 < item2 })

Como último parámetro de la función:

array.sort { (item1, item2) in return item1 < item2 }

Como último parámetro, usando nombres de argumentos simplificados (shorthand):

array.sort { return $0 < $1 }

Como último parámetro, con tipado implícito en el retorno:

array.sort { $0 < $1 }

Como último parámetro, como referencia a una función existente:

array.sort(<)
